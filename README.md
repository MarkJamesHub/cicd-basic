
# Add Gradle Wrapper to project

  Install dependancies: java-1.8.0-openjdk
  
    brew install...

  Install Gradle 6.0.1
  
    brew install gradle

  go to repo
  
    cd cicd-basic

  initialise the repo

    gradle init
    
    or
    
    gradle wrapper
  
  Edit build.gradle file (add some build tasks)
  
    vi build.gradle
    i
    <paste gradle.build.template content here>
    :wq!
  
  Run a gradle build
  
    ./gradlew build
